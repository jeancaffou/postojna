# Postojna Online

## [postojna.online](https://postojna.online)

This repository contains a static April Fools parody site presented in the style of satirical news outlets such as The Onion.

The project mimics the structure and visual language of a municipal news site, but the stories and announcements are intended as parody rather than factual reporting.

## Notes

- This is a satirical project for April Fools' Day.
- The content is not official municipal communication.
